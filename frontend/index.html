Backend Integration

Your F# backend should:

  1. Create customers using StripeService.createCustomer
  2. Create setup/payment intents and return client secrets
  3. Handle webhooks for payment confirmation
  4. Update your database when payments complete

Security

  • Never include secret keys in frontend code
  • Always validate payments on your backend
  • Use webhooks for reliable payment status updates
